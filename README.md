# clientes-web

Frontend en Angular para la gestion de clientes.

Para instalar las dependencias, ejecutar:

```bash
npm install
```

Para ejecutarlo en desarrollo, con el backend ya levantado, usar:

```bash
npm start
```

Luego acceder:

`http://localhost:4200`

Credenciales de prueba para desarrollo:

- Usuario: `admin`
- Contrasena: `Admin123*`

Estas credenciales son solo para fines de desarrollo y pruebas.
